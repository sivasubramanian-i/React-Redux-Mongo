## Installation

> client side installation

    ```
        $ npm install
    ```

> Start the client

    ```
        $ npm start
    ```

> Server side installation

    ```
        $ npm install
    ```

> Start the server

    ```
        $ npm start
    ```

> For Mongo (using Mongo Cluster)

    ```
       mongodb+srv://siva:siva12345@customer-cluster-b6ury.mongodb.net/test?retryWrites=true&w=majority
    ```
